### Why assembly?
because i was really bored and i wanted more of an excuse to procrastinate.

also: **aura**.

## Building
```sh
$ make day[1-12]
# example:
$ make day1
$ make day2
```

## Running
```sh
$ make run-day[1-12]
# example:
$ make run-day1
$ make run-day2
```
or manually, after building

```sh
$ ./build/day[1-12]
# example
$ ./build/day1
$ ./build/day2
$ ./build/day3
```
